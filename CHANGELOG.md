# Changelog

All notable changes to this project are documented here.

## [0.1.0] - 2026-07-14

### Added

- Rust CLI command structure for detection, info, reads, and field writes.
- Strongly typed MAC model, serial and MAC validation, binary record parser, and tests.
- Transport trait with safe mock device for development and UI integration.
- Write-readback verification path and actionable errors.
- Initial C ABI version entry point for future .NET P/Invoke integration.
- Architecture, API, and developer setup documentation.
